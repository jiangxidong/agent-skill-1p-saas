# Subniche Selection 评估模版

> [!abstract] 核心逻辑
> 在大市场中找到被风投忽视的垂直细分。目标是 $100K–$1M/年 的现金流业务，而不是追求独角兽。

---

## Basic Info

| 维度 | 填写 |
|------|------|
| **大市场** | [如：金融、房地产、教育、医疗] |
| **子利基** | [如：Gen Z FIRE 运动、本地屋顶公司] |
| **目标客户** | [职位/角色 + 公司规模 + B2B/B2C] |
| **内行程度** | [内行/半内行/外行 + 具体经验] |

---

## Market Attractiveness（5分制）

| 评估维度 | 分数 | 证据/理由 |
|----------|------|-----------|
| **Market Size** — 子利基内有足够客户支撑 $100K–$1M/年 | /5 | |
| **Willingness to Pay** — 客户有预算且习惯为软件付费 | /5 | |
| **Pain Intensity** — 当前工作流存在明显的时间/金钱浪费 | /5 | |
| **VC Blind Spot** — 没有风投支持的公司在正面竞争 | /5 | |
| **Automation Potential** — 大量机械性重复任务可用 AI 替代 | /5 | |

> [!tip] 评分标准
> - **18分以上** — 值得深入，立即启动验证
> - **12–17分** — 需要进一步验证，找5个潜在客户聊聊
> - **12分以下** — 建议换方向

## Personal Fit

| 维度 | 评估 |
|------|------|
| **Domain Knowledge** | 你对这个行业了解多少？能否和客户说同一种语言？ |
| **Technical Feasibility** | 以你的技术能力，MVP 需要多久？ |
| **Distribution Edge** | 你有没有现成的触达渠道？社媒、社区、人脉？ |
| **Personal Interest** | 你能坚持做这件事 1–2 年吗？ |

---

## Go / No-Go Decision

- [ ] Market Attractiveness >= 18 分
- [ ] 至少有 1 个明确的竞品弱点可以切入
- [ ] Personal Fit 至少 3 项为正面
- [ ] 能在 2 周内找到 5 个潜在客户聊一聊

> [!success] GO
> 满足以上全部条件，启动 Service-First 验证。

> [!danger] KILL
> Market Attractiveness < 12 或 Personal Fit 全部为负面。

> [!warning] PIVOT
> 部分条件不满足。调整方向：[描述]
